📅 Festival & Events Calendar

A stylish and interactive Monthly Calendar Web App that displays festivals and allows users to add their own events. Events automatically appear on the calendar with colored indicators and can be viewed in a sidebar. All added events are saved locally, ensuring they remain even after page refresh.

🌟 Features

📆 Interactive Monthly Calendar
Navigate between months using previous/next buttons.

🎉 Built-in Festival List
Popular Indian festivals are pre-loaded and shown automatically.

➕ Add Your Own Events
Users can add personal, festival, or other events for any date.

🌈 Stylish UI with Event Dots
Each day shows colored dots based on event type.

📝 Sidebar Event Viewer
Clicking a date shows all events for that day.

💾 Auto-Save with LocalStorage
User-added events stay permanently until deleted manually.

📱 Responsive Design
Works well on desktop, tablet, and mobile screens.
